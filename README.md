## 1. Variable

- Variable in C/C++

``` c++
int a = 1;
int b = 2.5; 
int c = 3, d = 4;
```

- Variable in Python

```python
a = 10
b = 1.5
```

- Variable in Java

```java
int a = 3;
int b = 8.2;
```

- Variable in C#

```c#
int a = 5.7;
int b = 3;
```

- Variable in PHP

```php
$a = 1; 
$b = 4.6;
```

- Variable in JavaScript

```javascript
var a = 8; 
let b = 9.3;
```
